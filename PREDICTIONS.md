# My predictions for *math.js*

Double - This function should be an elementary way to take a given number (x) and double it once, and return that result.

Square - This function should be an elementary way to take a given number (x) and multiply it by itself, i.e. square it, and return that result.

isEven - This function should check whether a number is even and output a message confirming or refuting that.

isOdd -  This function should check whether a number is odd and output a message confirming or refuting that.

multiply - This function should take two numbers and multiply them, then return the result.
